## 人工门禁

公开 release 打标签前，还需要在 Zotero 图形界面验证：

- 在 Zotero 9 当前稳定版运行 `doc/UI_REGRESSION_CHECKLIST.zh-CN.md`。
- 通过 Zotero `Tools -> Plugins` 安装 `.scaffold/build/zotero-cat.xpi`。
- 确认关闭并重开 Zotero 后设置仍保留。
- 确认打包安装后 API Key 仍能读取。
- 确认重启 Zotero 后会话历史仍保留。
- 声明 Zotero 10 兼容前，在当前 Zotero beta 上运行清单。
