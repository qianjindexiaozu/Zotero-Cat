# Zotero-Cat v0.1.2

[English](./0.1.2.md) | [中文](./0.1.2.zh-CN.md)

Zotero-Cat is a Zotero item-pane AI assistant for reading, summarizing,
reviewing, and discussing research items with user-selected model providers.

## Download

Download `zotero-cat.xpi` from the assets below and install it from Zotero's
Add-ons Manager.

Quick start:

1. Download `zotero-cat.xpi`.
2. Open Zotero.
3. Go to `Tools -> Plugins`.
4. Install the XPI file.
5. Configure an OpenAI-compatible provider in Zotero-Cat settings.

## Compatibility

- Supported: Zotero 9.x
- Not yet declared: Zotero 10 beta

Zotero 10 compatibility will be declared only after the current Zotero beta line
passes the manual UI checklist.

## Highlights

This is an engineering-quality patch release. No user-facing feature changes.

- Shared text utilities consolidated into `src/utils/text.ts`.
- Markdown-to-DOM rendering extracted from `section.ts` into
  `src/modules/agent/markdown.ts`.
- ESLint no-unused-vars changed from `off` to `warn`.
- Periodic expired-entry cleanup added for the selected-text cache.
- Zotero 10 `Components.Constructor` compatibility tracking added to the
  project phase plan.

## Upgrade Notes

Upgrade from `v0.1.1` simply by installing the new XPI. No configuration
migration is required. Conversation data and settings are preserved.

## Privacy

Zotero-Cat stores settings, custom context, and conversation data locally in
Zotero preferences. API keys are stored in Firefox Login Manager. Content is
sent only to the provider and optional search service configured by the user.

See:

- [Privacy](https://github.com/qianjindexiaozu/Zotero-Cat/blob/main/doc/PRIVACY.md)
- [隐私说明](https://github.com/qianjindexiaozu/Zotero-Cat/blob/main/doc/PRIVACY.zh-CN.md)

## Known Limitations

- Web search uses search snippets only; it does not crawl full webpages.
- Token budgeting is approximate.
- Model context length and reasoning-effort options depend on metadata returned
  by the provider.
- Zotero 10 beta support is not enabled yet.
- The assistant lives inside Zotero's item pane and shares space with Zotero's
  native item details.

## Non-affiliation
