# Zotero-Cat v0.1.1

[English](./0.1.1.md) | [中文](./0.1.1.zh-CN.md)

Zotero-Cat 是一个 Zotero 条目面板 AI 助手，用于阅读、总结、审阅和讨论文献条目，并支持用户自选模型提供方。

## 下载

从下方 assets 下载 `zotero-cat.xpi`，然后在 Zotero 的插件管理器中安装。

快速开始：

1. 下载 `zotero-cat.xpi`。
2. 打开 Zotero。
3. 进入 `Tools -> Plugins`。
4. 安装该 XPI 文件。
5. 在 Zotero-Cat 设置中配置 OpenAI-compatible provider。

## 兼容性

- 已支持：Zotero 9.x
- 尚未声明：Zotero 10 beta

只有在当前 Zotero beta 线通过人工 UI 检查后，才会声明 Zotero 10 兼容。

## 亮点

- 在 Zotero 原生右侧条目面板内提供聊天助手。
- 支持 OpenAI-compatible provider 和流式回复。
- 支持模型列表探测、上下文窗口解析，以及 provider 声明的 reasoning effort 解析。
- 可注入 Zotero 条目元数据、笔记、批注、选中文本和自定义上下文。
- 每个条目独立持久化会话，并支持重命名、收藏和导出。
- 可选联网搜索工具，支持 DuckDuckGo fallback 和可选 SearXNG。
- 支持解析模型输出的联网搜索 action，由 Zotero-Cat 执行工具，而不是依赖 provider-native tool calling。
- 扩大 prompt/context 预算，降低选中文本、笔记、批注和摘要被过早截断的概率。
- API key 使用 Firefox Login Manager 保存，不写入普通 Zotero prefs。
- 公开文档同时提供英文和中文版本。

## 升级说明

此版本取代早期 `v0.1.0-alpha` 预发布版本。该 alpha 包的 manifest version 是 `0.1.0`；`v0.1.1` 为已安装该预发布版本的用户提供真实包版本升级。

## 隐私

Zotero-Cat 会把设置、自定义上下文和会话数据保存在 Zotero prefs 本地。API key 通过 Firefox Login Manager 保存。内容只会发送到用户配置的模型提供方和可选搜索服务。

参见：

- [Privacy](https://github.com/qianjindexiaozu/Zotero-Cat/blob/main/doc/PRIVACY.md)
- [隐私说明](https://github.com/qianjindexiaozu/Zotero-Cat/blob/main/doc/PRIVACY.zh-CN.md)

## 已知限制

- 联网搜索目前只使用搜索结果摘要，不抓取完整网页。
- token 预算是估算值。
- 模型上下文长度和 reasoning effort 选项取决于 provider 返回的元数据。
- 尚未启用 Zotero 10 beta 支持。
- 助手位于 Zotero 条目面板内，会与 Zotero 原生条目信息共享空间。

## 非隶属声明

Zotero-Cat 是独立项目，不隶属于 Zotero。
