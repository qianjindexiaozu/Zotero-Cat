# Zotero-Cat v0.1.2

[English](./0.1.2.md) | [中文](./0.1.2.zh-CN.md)

Zotero-Cat 是一个 Zotero 条目面板 AI 助手，用于阅读、总结、审阅和讨论文献条目，模型提供方由用户选择。

## 下载

从下方资源下载 `zotero-cat.xpi`，在 Zotero 的 Add-ons Manager 中安装。

快速开始：

1. 下载 `zotero-cat.xpi`。
2. 打开 Zotero。
3. 进入 `工具 -> 插件`。
4. 安装 XPI 文件。
5. 在 Zotero-Cat 设置中配置 OpenAI-compatible 提供方。

## 兼容性

- 已支持：Zotero 9.x
- 尚未声明：Zotero 10 beta

Zotero 10 兼容性将在当前 Zotero beta 线通过人工 UI 清单后声明。

## 亮点

这是一个工程质量补丁版本，无用户可见功能变更。

- 共享文本工具函数合并到 `src/utils/text.ts`。
- Markdown 转 DOM 渲染从 `section.ts` 提取到 `src/modules/agent/markdown.ts`。
- ESLint `no-unused-vars` 从 `off` 改为 `warn`。
- 为选中文本缓存添加定期过期条目清理。
- 在项目阶段计划中添加 Zotero 10 `Components.Constructor` 兼容性跟踪。

## 升级说明

从 `v0.1.1` 升级只需安装新的 XPI。无需配置迁移。会话数据和设置会被保留。

## 隐私

Zotero-Cat 将设置、自定义上下文和会话数据本地保存在 Zotero preferences 中。API Key 保存在 Firefox Login Manager 中。内容仅发送给用户配置的模型提供方和可选的搜索服务。

参见：

- [Privacy](https://github.com/qianjindexiaozu/Zotero-Cat/blob/main/doc/PRIVACY.md)
- [隐私说明](https://github.com/qianjindexiaozu/Zotero-Cat/blob/main/doc/PRIVACY.zh-CN.md)

## 已知限制

- 联网搜索目前仅使用搜索片段，不抓取整页内容。
- Token 预算为估算值。
- 模型上下文长度和 reasoning-effort 选项取决于提供方返回的元数据。
- 尚未开启 Zotero 10 beta 支持。
- 助手位于 Zotero 条目面板内，与 Zotero 原生条目详情共享空间。

## 非隶属声明

Zotero-Cat 是独立项目，不隶属于 Zotero。
