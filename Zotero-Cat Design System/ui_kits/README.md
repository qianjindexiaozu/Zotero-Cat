# Zotero-Cat — UI kits

Two surfaces:

- `plugin/` — the Zotero item-pane chat UI (compact, 360px tall, runs inside Zotero's right sidebar).
- `web/` — the marketing site (hero + terminal + feature triad + 4-step quickstart + provider chips + story block).

Each folder has its own `index.html` to demo the kit, plus standalone HTML files for individual sections so they can be lifted into prototypes.
