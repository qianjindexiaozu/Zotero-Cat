# Plugin UI kit · `Zotero-Cat` item-pane

Single screen recreation of the Zotero right-pane chat UI.

- 360px tall, 380px wide frame
- Streaming bubbles + composer + context controls + diagnostics
- Inherits Zotero's text color via `color-mix(in srgb, currentColor X%, transparent)`
- See `ui_kits/plugin/index.html` for the demo
